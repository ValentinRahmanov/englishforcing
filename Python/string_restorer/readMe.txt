ДЕКОДЕР СТРОК, ЗАШИФРОВАННЫХ МЕТОДОМ КОДИРОВАНИЯ ПОВТОРОВ
1. Метод кодирования повторов заключается в том, что повторяющиеся символы в строке заменяются числом повторов. Тем самым строку удается сжать для более эффективного хранения.

Например, строка:

aaabbbccc

таким образом, строка приобретает более удобный вид:

a1b1c1

3. Этот скрипт на вход получает сжатую таким способом строку в файле compressed_string.txt

На выходе получаем расшифрованную строку, записанную в файл result.txt
